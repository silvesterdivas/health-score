02 ‚Äî How It Works

A non-linear curve,
because decay isn't linear.

The score maps context fill % to a health value using a two-segment curve: gentle decay from 100‚Üí50 through the first 73%, then accelerated collapse to 0. This mirrors how LLM quality actually degrades.
